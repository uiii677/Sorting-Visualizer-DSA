**DSA Sorting Visualizer** is a web-based application designed to visually demonstrate how different sorting algorithms work step-by-step.  
The project helps students understand the internal working of sorting algorithms through animations, comparisons, and real-time execution.

This visualizer allows users to:
- Select different sorting algorithms
- Control array size and speed
- Observe comparisons and swaps visually
